#Bitácora del Explorador Estelar

Este proyecto implementa una **bitácora de eventos** utilizando una **Pila (Stack)** en Java.  
La bitácora simula el registro de sucesos dentro de una nave espacial, permitiendo consultar y eliminar eventos siguiendo el principio **LIFO (Last In, First Out)**.

Funcionamiento
Registrar eventos: se guardan sucesos en la bitácora (ejemplo: encendido de motor, anomalías, errores críticos).
Consultar último evento: se obtiene el evento más reciente registrado.
Protocolo de revisión: si el último evento contiene un ERROR, se eliminan los últimos tres eventos como medida de seguridad.
Estado actual: se muestra cuántos eventos quedan y cuál es el último disponible.
